Guidelines

Usage and best practices

Getting Started

EcoKit is the design system for the Open Climate Transparency Platform. Use these guidelines to ensure consistent implementation across all applications.

Accessibility

All components follow WCAG 2.1 AA guidelines. Ensure proper:

  • Color contrast ratios
  • Keyboard navigation
  • Screen reader support
  • Focus indicators

Best Practices

  • Use design tokens for colors, spacing, and typography
  • Follow component API patterns consistently
  • Test components across different devices and browsers
  • Maintain semantic HTML structure
